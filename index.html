Realice un programa que solicite un número 'N', luego calcule y muestre el factorial de este número, por último, pregunte si desea salir, si no desea salir vuelva solicitar el número 'N' para el calculo factorial, caso contrario termine el programa.

// Factorial de un numero
int n, f;
string respuesta;
Console.WriteLine("Número faltorial");
do 
{ 
    Console.Write("\nIntroduzca el numero? ");
    n = int.Parse(Console.ReadLine());
    if (n == 0)
        Console.WriteLine("El factorial de {0} es = 1", n);
    else
    {
        f = n;
        for (int i = n; i > 1; i--)
            f = f * (i - 1);
        Console.WriteLine("El factorial de {0} es = {1}", n, f);
    }
    Console.Write("\n¿Desea salir? [S/n] ");
    respuesta = Console.ReadLine();
} while (respuesta == "n" || respuesta == "N");
